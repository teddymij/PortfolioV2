Mes compétences

Deployer un site web
  • Services de cloud : Savoir utiliser des services de déploiement cloud comme Heroku, Netlify, Vercel, AWS, Google Cloud, ou DigitalOcean.
  • Configuration de serveurs : Pour un déploiement sur un serveur, comprendre l’utilisation de serveurs Nginx ou Apache pour héberger l'application et la configuration des environnements de production.
  • Déploiement continu (CI/CD) : Configurer des pipelines de déploiement avec des outils comme GitHub Actions, GitLab CI, ou CircleCI.
Developper une application web
  • HTML/CSS : Connaître la structure des pages web (HTML) et le style (CSS). Utilisation des Flexbox, Grid, animations et transitions.
  • Frameworks JavaScript : Utiliser des frameworks comme React, Vue.js, Angular, etc., pour le front-end.
  • Préprocesseurs CSS : Utiliser Sass ou Less pour un CSS plus modulaire et facile à maintenir.
  • Backend : Connaître un langage backend (ex : Node.js, PHP, Python/Django, Ruby on Rails) et savoir interagir avec une base de données.
Versioning et Collaboration :
  • Git : Utilisation de Git et GitHub pour la gestion des versions, travailler en équipe, et gérer les branches.
  • Gestion de projet : Utiliser des outils de gestion de projet (comme Trello, Asana, Jira) pour organiser les tâches et le travail d'équipe.
Sauvegarde et sécurité
  • Backup régulier : Assurer des sauvegardes régulières des données et du code.
  • Sécurisation des accès : Limiter les accès au serveur (utilisation de clés SSH pour l’accès sécurisé).
  • Logs et suivi : Mettre en place des outils pour suivre les logs de l’application et la gestion des erreurs (ex. Sentry, LogRocket).
  • Monitoring de performance : Utiliser des outils de suivi des performances comme Google Analytics, New Relic, Datadog.
